## Overview
This project analyzes the wealth, criminal records, education profile, demographic trends, and representation patterns of Karnataka MLAs from the 2023 State Assembly Elections.

The project focuses on exploratory data analysis, dashboard storytelling, and insight generation using Power BI, SQL, Excel, Power Query, and DAX.

## Key Insights

- 54.9% of MLAs have criminal cases
- ~33% face serious criminal charges
- Top 10 MLAs account for ~46% of total declared wealth
- Criminal cases exist across both low and high wealth groups
- Majority of MLAs fall within the 50–65 age group
- Women representation remains significantly low
