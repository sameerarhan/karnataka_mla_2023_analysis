# Karnataka MLA Analysis (Power BI Dashboard)

## Overview
This project analyzes the wealth, criminal records, and education profile of MLAs from Karnataka (2023 State Assembly Elections).


## Key Insights
- 54.9% MLAs have criminal cases
- Top 10 MLAs hold ~46% of total wealth
- Criminal cases exist across both low and high wealth groups
- Majority of MLAs fall in 50–65 age group

## Tools Used
- Power BI (DAX, Power Query)
- Excel
- SQL

## Dashboard Features
- Interactive slicers
- Wealth segmentation
- Criminal case analysis
- Education vs wealth insights

## Data Sources
- myneta.info
- ECI
- ADR reports
